Релиз Yii2-user 0.3.0

Рад сообщить, что сегодня я выпустил новую версию модуля для работы с пользователями в Yii2. С момента последнего релиза прошел месяц, и за это время было сделано достаточно много.

Ниже список наиболее важных изменений:

  • Добавлены консольные команды
  • Добавлена простая регистрация (когда пароль генерируется автоматически)
  • Отрефакторены тесты
  • Добавлены интерфейсы для модели User
  • Отрефакторены формы
  • Добавлена опциональная каптча на все важные формы

В дальнейшем модуль продолжит развиваться, среди основных планов: управление аккаунтом, управление пользователями, авторизация через социальные сети.

Из того, что модуль умеет сейчас можно выделить:

  • Регистрация с опциональным подтверждением почты
  • Авторизация
  • Сброс пароля

+ покрытие юнит и функциональными тестами с использованием Codeception.

Для установки расширения необходимо выполнить команду

$ php composer.phar require dektrium/yii2-user "~0.3@dev"

или добавить

{
    "require": {
        "dektrium/yii2-user": "~0.3@dev"
    }
}

в composer.json.
